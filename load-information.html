Firmware Load Information

The firmware downloaded by the phone from the provisioning server is specified by <loadInformation> in SEPMAC.cnf.xml without the .loads suffix. To download firmware updates you will first need to register for a login (free, at the time of writing) at www.cisco.com. Firmware updates are selected via the Session Initiation Protocol (SIP) Software link when downloading software.

To convert firmware packages from .cop.sgn to .tar.gz use the stripsgn script.
